Тема: Разработка приложения для анализа и прогнозирования значений волатильных временных рядов. Это приложение позволяет загрузить ваши данные (волатильный временной ряд) и по ним провести анализ, построить графики и обучить на нмх модель, для прогнозирования значений.
Пользователи: тысячи разработчиков и аналитиков
Требования:
- Возможность загрузки данных в формате .csv
- Возможность проанализировать данные или выбранную часть данных
- Возможность обучить модель машинного обучения на загруженных данных для прогнозирования значений
- Возможность построить графики
Дополнительный контекст:
- Приложение ориентировано именно на волатильные временные ряды, то есть последовательность значений изменяющихся во времени и имеющих нестабильные значения
